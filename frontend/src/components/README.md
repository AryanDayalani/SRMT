# Components Directory

This directory contains reusable UI components for the Research Management Tool.

## Component Structure

- `layout/` - Layout components like Sidebar, Navbar, etc.
- `ui/` - Basic UI components like Button, Card, etc.
- `forms/` - Form components for data input
- `dashboard/` - Dashboard specific components
- `auth/` - Authentication related components
- `projects/` - Project management components
- `paper-analyzer/` - Paper analysis components
- `collaborators/` - Collaborator management components
- `notifications/` - Notification componentsgit add .
